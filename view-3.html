COVID-19 Map


  • This choropleth map uses the total number of positive cases by state between January 13, 2020 and March 7, 2021. California, Texas, and Florida have the highest numbers of positives.
  • This map was made using amCharts 5 in Javascript with the Covidtracking.com API historical dataset.
  • Use: When any state is clicked, the map will zoom in and display the total number of positives for that state within the time frame.
  • Limitations: this dataset does not include updated Covid-19 statistics.
